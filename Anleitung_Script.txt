
Anleitung: Nutzung von combine_participant_files.R

1. Installiere R & RStudio (App zur Verwendung von R)
→ Installiere R zuerst: https://cran.r-project.org/
→ Installieren R Studio: https://posit.co/download/rstudio-desktop/

2. Lege deine Daten im Ordner data/ ab
→ Alle Teilnehmerdaten als .xlsx speichern
→ (optional: Script kann auch für .csv angepasst werden)

3. Öffne das R-Script und führe es aus
→ Gesamten Code mit der Maus markieren
→ Mit Strg + Enter ausführen

4. Ergebnis:
→ Eine kombinierte Datei df_combined.sav wird im Hauptordner erstellt. Diese kann mit SPSS eingelesen werden.
→ (optional: Umstellung auf .xlsx möglich mit write_xlsx())

⸻

Hinweise
- Jeder Schritt im Code ist durch Kommentare (#) erklärt
- Teile des Codes sind aus-kommentiert und können bei Bedarf durch das Löschen von "#" verwendet werden.
- Bei Problemen: googlen, nachfragen oder ChatGPT nutzen (liest auch Fehlermeldungen)
